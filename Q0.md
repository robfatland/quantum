Let's take as a *given* that there exist some familiar subatomic particles: electrons, positrons, neutrons, protons, photons, helium nuclei (2 protons + 2 neutrons; a single 'alpha' particle), muons; and so on. We also have notions of exotic particles like neutrinos and dark matter. But for these actors to stage a meaningful play we need some sort of stage; so this stage we shall call according to tradition *Hilbert space* and we'll begin by defining it via stages of context.  


Paul Dirac was there at the bar and at one point he made this remark on the relationship between mathematics and physics:


- **The mathematician plays a game in which he himself invents the rules while the physicist plays a game in which the rules are provided by nature, but as time goes on it becomes increasingly evident that the rules which the mathematician finds interesting are the same as those which nature has chosen.**


In the mathematical field of *analysis* we begin with a set $S$, a collection of distinct elements. Add two operations *addition* and *multiplication*: They may be applied between any two elements of the set. $S$ is said to be *closed* under addition and multiplication if all sums / products are also in $S$. 


Now suppose with S closed under addition we add four axioms. First for all $a$, $b$, $c$ in $S$ we have $(a + b) + c = a + (b + c)$. This is **associativity**. Second there is an **identity element** called $0$ such that for all $a$ in $S$ we have $a + 0 = a$. Third we need additive inverses: For every $a$ in $S$ there is $-a$ also in $S$ such that $a + (-a) = 0$. Fourth we say addition is **commutative**: For every $a$ and $b$ in $S$ we have $a + b = b + a$. Together with closure all this means that $S$ is an abelian group under addition. So far so good. 


Now for multiplication: $(a \cdot b) \cdot c = a \cdot (b \cdot c)$ is associativity. The multiplicative identity let us call $1$. We would want multiplicative inverses for all $a$ in $S$ and here we hit a snag because the $0$ element we mentioned above has no multiplicative inverse. So we refine the second 'multiplication' abelian group by stipulating all elements of $S$ except $0$. (Multiplication commutes in abelian groups.)


We have abelian groups working for $(S, +)$ and for $(S \backslash \{0\}, \times)$. Finally one more axiom is needed that relates addition and multiplication, the distributive axiom: $a(b + c) = a \cdot b + a \cdot c$. (Technically we should add $(b + c)a$ for both *left* and *right* distributivity. 
This combination of +-abelian, x-abelian and the distributive axiom 
brings us to a **field** $F$. We need this field formalism to provide us with *scalars* in what follows. 


A *vector space* $V$ over a field $F$ is a set of elements called 'vectors' that has two operations defined; where these two operations follow certain rules. The first operation is vector addition; so it applies between two elements of $V$, as in  $u + v$. The second operation is 'scalar multiplication' for some scalar $a \in F$. (Technically scalar multiplication is a *map* rather than an *operation*, symbolically $F \times V \to V$.) 


The vector space $V$ must satisfy these rules:


- $(V, +)$ is an abelian group: Associativity, Identity, Inverses, Commutativity
- Scalar multiplication $a \cdot \tilde{v}$ follows these four axioms
    - Compatibility with field multiplication: $a(bv) = (ab)v$
    - $F$ identity operates trivially: $1 \cdot v = v$
    - Distributivity over vector addition: $a \cdot (u + v) = au + av$
    - Distributivity over field addition: $(a + b)v = av + bv$


Notice the abelian group with vector addition echoes the case for the field $F$. Similarly the Compatibility and Distributivity axioms echo the distributive axiom for $F$. 


Here is where practical use of vectors (say in linear algebra) often muddles the abstraction.
A vector $v \in V$ is an abstract entity that obeys the rules of the vector space. There is a
thought process by which we can produce something called a *basis*, a set of vectors in $V$
that accomplish a task of *representation*. I'll define this below; but the point is that
a particular basis is not unique. It can be abandoned in favor of a *different* basis, and so
on potentially to vastly many *other* bases. So a basis is a formal structure imposed on a
vector space that allows us to express arbitrary vectors in $V$ by means of representation.
