# quantum

Let's investigate quantum computing on various cloud platforms. These are broken into folders by provider.

## Black Holes

The `blackholes/` directory contains an extended dialog on black hole physics (Hawking radiation,
evaporation timescales, greybody spectra, the information paradox) plus an observational cosmology
script that connects dark energy evidence to the long-term fate of black holes.

- `blackholes.md` — Dialog covering Hawking radiation, particle species, evaporation, negative heat capacity, information paradox
- `dark_energy_analysis.py` — Pantheon+ Type Ia supernovae analysis demonstrating accelerating expansion
- `Pantheon+SH0ES.dat` — Dataset for the analysis script
